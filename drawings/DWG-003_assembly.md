## 4. 配線接続図

### 4.1 電気配線 (Zone C → 各Zone)

```
DC入力(28V) → DC-DC → 3.3V (FPGA core)
                     → 1.8V (FPGA I/O)
                     → 5V (ADC, DAC, PD TIA)
                     → 12V (Peltier, EO driver)
                     → ファン (PWM制御, FPGA GPIO)

Zone C → Zone A: EO driver RF同軸 ×2 (SMA shielded, ~100MHz)
                  PZT driver DC ×2 (ツイストペア shielded)
                  TEC制御線 ×2 (シールド付き)
```
