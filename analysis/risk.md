## 1. CV方式の成功基準定義

DV-FBQC方式のレベルA/B/Cに対応するCV方式の成功基準:

| レベル | DV定義 | **CV定義** |
|-------|--------|----------|
| **A（スケーラブルQEC）** | erasure < 閾値 | **p_L < 10⁻⁵ (d=7, MWPM), σ_eff ≥ 9.7dB** [v3.2: 旧10⁻⁶は理論限界L=0.15dBでのみ達成可能。製品要件は10⁻⁵] |
| **B（エラー抑制実証）** | d=3→d=5で改善 | **d=3→d=5でp_Lが10倍以上低減** |
| **C（コンポーネント実証）** | 各要素が仕様通り | **OPAスクイージング>10dB, GKP Wigner確認** |

### 2.2 レベル別成功確率

**確率算出方法**: Level Aは全要素の成功が必要ではなく、クリティカルパス上の要素のみ。
WDMとsoft-info MWPMは独立に達成可能で、いずれかが失敗しても性能低下で対応可。

| レベル | 必要条件 | **CV pure確率** | **CV+QD確率** | DV v5.0 (参考) |
|-------|---------|:--------------:|:------------:|:--------------:|
| **C（コンポーネント）** | OPA>10dB + TDM動作 | **95%** | 95% | 80% |
| **B（エラー抑制）** | GKP生成 + d=3 break-even | **70%** | 80% | 55% |
| **A（スケーラブルQEC, MWPM）** | p_L<10⁻⁵(MWPM), d=7, PIC | **50%** | 65% | 12-18% |
| **A（スケーラブルQEC, 保守的）** | p_L<10⁻⁵(保守), d=7必須 | **35%** | 55% | 12-18% |

**注 (v3.2: 製品要件10⁻⁵ + 現実的L=0.27dB反映)**:
- **Phase 2+ PIC理論限界 (L=0.15dB)**: MWPM d=7で p_L≈5.7×10⁻⁷、UF d=7で6.7×10⁻⁶ → 両方とも10⁻⁵達成。
- **Phase 2+ PIC現実的 (L=0.27dB)**: MWPM d=7で p_L≈2-5×10⁻⁵(境界)、UF d=7で~5×10⁻⁴(10⁻⁵超過) → **MWPM必須**。
- Phase 1(離散光学, 13dB, L=0.39dB): MWPM d=7で p_L≈1.6×10⁻³ → Level A未達、break-even実証レベル。
- 製品級FTQC(p_L<10⁻⁵)にはPhase 2+ PIC統合(L≤0.27dB)+MWPM d=7が必須。L=0.15dBで余裕確保。

### 3.1 最悪ケース分析

全リスクが同時に顕在化した場合:
- σ_eff: Phase 2+ PIC基準10.9dB - 位相ロック不安定劣化 - OPA低下 → 最悪ケース~9.0dB (v2.0 BSモデル。旧値11.5dBはdB減算公式による誤り)
- p_L(d=7, UF) ≈ 10⁻² → break-even未達

**最悪ケースでもσ_eff = 9.0dB。** Stafford-Menicucci postselection閾値7.5dBに対しマージン+1.5dB。
QECは動作するがLevel A未達。Level B(break-even)にはd=5で十分（σ_eff>8.5dBで達成）。

**12dBフォールバック時 (v2.1 BSモデル)**: Phase 2+ PIC(L=0.15dB)で12dB生成:
V_eff = 0.966×10⁻¹·²+0.034 = 0.966×0.063+0.034 = 0.095 → σ_eff≈10.2dB → p_err≈1.7×10⁻³
MWPM d=7: p_L≈4.5×10⁻⁶ → Level A未達だがLevel B達成。
QD追加(+2.7dB等価)でLevel A回復可能。旧dB減算式(10.5dB)は不正確。

### 5.1 相関失敗分析

上記の>95%推定は各構成の失敗が独立であると仮定している:
P(少なくとも1つ成功) = 1 - (1-P_CV_pure) × (1-P_CV_QD) × (1-P_DV)

しかし、共有失敗モード（correlated failure modes）が存在するため、独立仮定は楽観的である。

#### 相関する失敗要因

| 共有失敗モード | 影響範囲 | 失敗確率 | 備考 |
|--------------|---------|---------|------|
| **NTT OPA供給障害** | CV pure + CV+QD（両方） | ~20% | PPLNウェハ品質・供給ライン単一依存 |
| **PPLN品質劣化** | CV pure + CV+QD（両方） | ~15% | NTT OPAと部分的に重複 |
| **人材不足（CV専門家）** | CV pure + CV+QD（両方） | ~10% | CV量子光学の人材プール小 |
| **GKP postselection原理不成立** | CV pure + CV+QD（両方） | ~25% | 光学GKP未実証。失敗時CV全滅、DVのみ存続 |

#### 相関を考慮した再計算

**NTT OPA依存を主要相関因子としてモデル化:**

- P(NTT OPA正常供給) = 0.80
- P(NTT OPA供給失敗) = 0.20

**Case 1: NTT OPA正常 (確率0.80)**
- P(CV pure レベルB以上) = 0.70
- P(CV+QD レベルB以上) = 0.80
- P(少なくとも1つのCV成功) = 1 - (1-0.70)×(1-0.80) = 1 - 0.06 = 0.94
- P(DV レベルB以上) = 0.55
- P(少なくとも1つのFTQC) = 1 - (1-0.94)×(1-0.55) = 1 - 0.027 = 0.973

**Case 2: NTT OPA供給失敗 (確率0.20)**
- P(CV pure) = 0（OPA依存のため全滅）
- P(CV+QD) = 0（同上）
- P(DV レベルB以上) = 0.55（DVは独立）
- P(少なくとも1つのFTQC) = 0.55

**追加相関因子 (v1.1追加, v2.2ベイジアン修正): GKP⇔OPA物理的従属性**

CV pure と CV+QD は GKP postselection の物理原理を共有する。さらに**GKP成功確率はOPA品質に物理的に従属**する（σ_gen低下→σ_eff低下→F_GKP低下→postselection確率低下の連鎖）。独立事象としての確率乗算は不正確。

**ベイジアン条件付きモデル (v2.2)**:
- P(OPA≥13dB) = 0.70（NTT SiN clad改良成功）
- P(OPA=12.1-12.9dB) = 0.25（現状維持〜微改善）
- P(OPA<12.1dB) = 0.05（装置故障レベル）

- P(GKP成立|OPA≥13dB) = 0.80（σ_eff十分、F>0.85達成見込み高）
- P(GKP成立|OPA=12.1-12.9dB) = 0.40（σ_eff不足でF<0.85リスク大）
- P(GKP成立|OPA<12.1dB) = 0.0（原理的に閾値未達）

**Case A (OPA≥13dB × GKP成立, 確率 0.70×0.80=0.56)**:
- P(CV pure) = 0.70, P(CV+QD) = 0.80 → P(CV成功) = 0.94
- P(DV) = 0.55 → P(FTQC) = 1-(1-0.94)(1-0.55) = 0.973

**Case B (OPA≥13dB × GKP不成立, 確率 0.70×0.20=0.14)**:
- P(CV) = 0, P(DV) = 0.55 → P(FTQC) = 0.55

**Case C (OPA=12.1-12.9dB × GKP成立, 確率 0.25×0.40=0.10)**:
- P(CV Level B) = 0.50（break-even可能だがLevel A未達）, P(DV) = 0.55
- P(FTQC Level B以上) = 1-(1-0.50)(1-0.55) = 0.775

**Case D (OPA=12.1-12.9dB × GKP不成立, 確率 0.25×0.60=0.15)**:
- P(CV) = 0, P(DV) = 0.55 → P(FTQC) = 0.55

**Case E (OPA<12.1dB, 確率 0.05)**:
- P(CV) = 0, P(DV) = 0.55 → P(FTQC) = 0.55

**v2.2修正統合確率**:
P(FTQC) = 0.56×0.973 + 0.14×0.55 + 0.10×0.775 + 0.15×0.55 + 0.05×0.55
        = 0.545 + 0.077 + 0.078 + 0.083 + 0.028 = **0.811**
P(Level A FTQC) = 0.56×0.94×0.6 + ... ≈ **~63%** (旧80%から下方修正)

注: v2.2ベイジアン修正により、OPA⇔GKP物理的従属性を反映。旧モデル(独立仮定)の80%は過大推定。
ただしPhase -1 T0b/T0cでGKP Go判定(F>0.80, Δ<0.15)確認後は
P(GKP成立|OPA≥13dB)が0.80→0.92に上昇し、P(FTQC)は~70%に回復する。

#### 結論

**v2.2修正**: OPA⇔GKP物理的従属性を考慮した成功確率は **~63% (Level A)** / **~81% (Level B以上)**。
旧値80%(v1.1)は独立仮定による過大推定。GKP原理リスク+OPA品質リスクが主要因。
OPA供給冗長化で改善にはOPA供給元の冗長化が必要:

- **対策1**: 自社PPLN開発（長期、2027Q4目標）
- **対策2**: 代替OPA供給元の確保（e.g., Covesion, HC Photonics）
- **対策3**: Option B パルスOPA方式をNTT非依存で設計（G-CV1 No-Go時の代替パス。旧CW OPA+EO gateはv2.0で非推奨）
