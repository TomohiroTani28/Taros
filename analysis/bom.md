## 5.5 製造歩留まり・スクラップ予算 (v2.1追加)

**100台/年量産時の歩留まり見積もり**:

| 工程 | 歩留まり | 根拠 | リスク対策 |
|------|----------|------|----------|
| PPLN OPA選別 | **80-90%** | NTT CW 12dB以上品選別。Phase -1 T0aで実測。10-20%が11dB未満で不合格 | FC/APCモジュール化により不合格品を即時スワップ(追加コスト75万円/台) |
| 融着接続 | **95%** | 16箇所x0.02dB、累積<0.35dB必達。5%は再スプライス | 作業者2名体制+自動スプライサ(約1,200万円設備) |
| FPGA + 電子系 | **98%** | 市販品。初期不良率<2%(標準) | 48hバーンイン選別 |
| システム統合テスト | **90%** | σ_eff≥10.4dB合格（Phase 2+ PIC: 10.9dB設計、0.5dBマージン。v2.0 BSモデル準拠）。10%はPLL調整再実施 | 再較正8hで復帰可能（スクラップではない） |
| **システム歩留まり** | **約67%** (初回パス) | 0.85x0.95x0.98x0.90=0.67 | |
| **リワーク後最終歩留まり** | **約92%** | 不合格33台中75%がリワーク復帰(OPAスワップ+再較正) | 8台/年は真のスクラップ(σ_eff回復不能) |

**コスト影響**:

- リワーク費用: 25台x75万円(OPA)+8h較正(12万円) = **約2,175万円/年**
- スクラップ損失: 8台x1,200万円 = **約9,600万円/年**
  （ただし部品回収で50%回復 → 実損約4,800万円）
- **100台出荷に必要な投入: 約109台分の部材** (9%オーバービルド)
- BOM原価修正: 約1,125万円 → 約1,125万円 + 約22.5万円(リワーク按分) + 約48万円(スクラップ按分) = **約1,215万円/台実効原価**
  （v2.1: ポンプ60万円→150万円反映）
- 販売価格1,800万円に対し粗利: (1,800万円-1,215万円)/1,800万円 = **32.5%**
  （目標40%に対し-7.5pt。量産スケール拡大またはSOA/SHGコスト低減で改善見込み）
- 参考: Phase 2量産1000台で原価約885万円達成時 → (1,800万円-885万円)/1,800万円 = **51%**
- 研究機器市場(Thorlabs/Keysight等)では粗利30-40%が標準的であり38%は適正

**Phase -1検証項目**: T0a OPA 20本評価で歩留まり実測 → 80%以上でGo、60%未満で代替ソース前倒し。
