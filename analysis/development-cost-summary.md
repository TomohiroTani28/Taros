# デスクサイド光量子コンピュータ Taros — 開発費用と計画

**Document ID**: PQC-COST-SUMMARY-v1.0
**Last Updated**: 2026-05-05

---

## はじめに

Tarosは「電源コンセントとLANケーブルだけで動く量子コンピュータ」を目指すプロジェクトです。超伝導方式（IBM、Google）が数十億円規模のmK冷凍機設備を必要とするのに対し、Tarosは光の性質を利用して室温で動作するため、桁違いに低コストで開発できます。

本文書では、Tarosの開発に必要な費用、期間、各段階で何が得られるかをまとめます。

---

## 開発の全体像

開発は4つのPhaseに分かれ、各Phase終了時にGo/No-Go判定を行います。判定結果が悪ければそこで止められるため、全額をリスクに晒す必要はありません。

|-------|------|------|------|--------|
| Phase -1: 原理検証 | 12ヶ月 | $3.04M (4.6億円) | $3.04M | 設計の成立/不成立を実験で確定 [v3.1: 8→12ヶ月延長確定] |
| Phase 0: 試作機 | 12-18ヶ月 | $1.2M (1.8億円) | $4.24M | 世界初のQEC break-even達成 |

## 各Phaseの詳細

### Phase -1（2026年5月-2027年4月）: 原理検証 — $3.04M [v3.1確定]

v3.1確定: 12ヶ月（GKP実験10ヶ月のクリティカルパス + T10/T11の8ヶ月超過）。予算$3.04M（タスク$1.16M + 人件費$1.92M）。

このPhaseの目的は「本当に作れるのか？」に最終回答を出すことです。

**やること:**
- PPLN導波路OPAで12dB以上のスクイージング光を生成する実験
- GPUシミュレーションで誤り訂正の閾値を精密検証
- GKP量子状態の光学的生成実験（世界初クラス）
- 位相ロック制御系の設計と基礎実験

**費用の内訳:**
- チーム11名の人件費（12ヶ月）: $1.92M（$160K/月。詳細は `analysis/phase-minus1-execution.md` §5.1）[v3.2: 旧8ヶ月→12ヶ月]
- タスク投資（装置・設備・委託）: $1.16M（新規$0.61M + 既存設備活用$0.55M。内訳は§5.2）
- 合計: $3.08M → **~$3.04M**（丸め）[v3.2修正]

**得られるもの:**
- 「作れる」or「作れない」の明確な判定データ
- 作れる場合: 投資家・パートナー向けの実験実証データ
- 作れない場合: $3.04Mでプロジェクト終了（被害限定）[v3.3: 12ヶ月版に修正]

**やること:**

## なぜこの金額で可能なのか

量子コンピュータの開発に通常数百億円かかる理由と、Tarosが桁違いに安い理由を説明します。

### 超伝導方式（IBM/Google）が高い理由

| 必要なもの | 費用 | Tarosでは |
|-----------|------|-----------|
| 希釈冷凍機（15mK） | $3-5M/台 | **不要**（室温動作） |
| クリーンルーム fabrication | $50-100M | **不要**（ファイバ光学系） |
| 超伝導配線・制御装置 | $10-50M | **不要** |
| 大規模冷却設備 | $5-10M | **不要** |

### Tarosが安い理由

1. **冷凍機が不要**: 1550nm光子は室温で熱雑音ゼロ（n_th = 3×10⁻¹⁴）
2. **光ファイバで構成**: 半導体fab不要。既製品を接続するだけ
3. **少人数で実現可能**: 設計の複雑度が低い（部品数が少ない）
4. **NTTの既存技術を活用**: PPLN導波路OPAは共同研究で入手可能

## リスクと段階的判定

全額を一度に投じるのではなく、各段階で成否を判定します。

### 最小投資での判定

| 投資額 | 得られる情報 | 期間 |
|--------|------------|------|
| **$5K（75万円）** | 設計が原理的に成立するか（GPUシミュレーション） | 2週間 |
| **$58K（装置直接費。人件費込み完全コスト: $198K）** | GKP光学実験：スクイージング光が出るか | 5ヶ月 | [v3.2: 旧3ヶ月→5ヶ月、$177K→$198K]
| **$95K（1,400万円）** | 投資家向けデモ可能なデータ一式 | 5ヶ月 |
| **$370K（5,600万円）** | QEC break-even実証（世界初） | 12ヶ月 |

**75万円で「やる価値があるか」がわかり、600万円で「光が出るか」が確認でき、5,600万円で「世界初の成果」が出ます。**

**Phase -1 費用定義の明確化（3段階表記）**:
- **$5K（最小判定コスト）**: Stimシミュレーションで設計原理の成立/不成立を判定
- **$58K（装置直接費）**: GKP実験装置の直接費用（G-EXP1用光学部品+計測器）
- **$198K（完全コスト）**: $58K装置費 + 人件費2名×5ヶ月込みGKP実験の完全コスト [v3.2: 旧$177K→$198K。修正タイムライン(5ヶ月)基準]
- $319K: CV固有技術検証の全装置費用（$58K含む）+ 基盤設備$280K
- $3.04M: 人件費込みPhase -1全体予算（11名×12ヶ月$1.92M + タスク投資$1.16M = $3.08M≈$3.04M）[v3.2: 旧8ヶ月→12ヶ月]
投資家向けには$3.04Mが正確なPhase -1総額。$5K→$58K→$177Kの3段階でリスクを段階的に評価可能。

※GKP実験費用の定義: 装置BOM $42K (OPA×2正確計上) / 装置直接費 $58K (予備・消耗品込み) / 完全コスト $198K (人件費2名×5ヶ月込み、修正タイムライン基準)

### 失敗した場合の損失限定

| 失敗ポイント | その時点の投資 | 理由 |
|-------------|-------------|------|
| Stimシミュレーションで閾値不足 | $5K | 理論的に不成立 → 即中止 |
| OPAスクイージング不足 | $58K（装置直接費。人件費込み完全コスト: $198K） | NTT品質問題 → 改良待ち |
| GKP生成不可 | $95K | 物理的制約 → 方式変更 |
| TDM不安定 | $370K | 位相ロック問題 → 技術改良 |

## 競合との比較

| 会社 | 累計調達額 | 現在の状態 |
|------|-----------|-----------|
| PsiQuantum | $700M+（1,000億円超） | 未実証、fab建設中 |
| Xanadu | $250M+（375億円） | Borealis（QECなし） |
| IonQ | $600M+（900億円） | 32 qubit、非FTQC |
| Google | $数B | 105 qubit、break-even実証 |
| **Taros（計画）** | **$5.74M（8.6億円）** | **QEC break-even → 量産** | [v3.2修正]

Tarosの戦略は「小さく始めて、実証してから拡大する」です。約8億円で競合が数百億円かけても達成できていない成果（デスクサイドFTQC）を狙います。
