# デスクサイド光量子コンピュータ Taros — 開発費用と計画

**Document ID**: PQC-COST-SUMMARY-v1.0

---

## はじめに

Tarosは「電源コンセントとLANケーブルだけで動く量子コンピュータ」を目指すプロジェクトです。
超伝導方式（IBM、Google）が数十億円規模のmK冷凍機設備を必要とするのに対し、
Tarosは光の性質を利用して室温で動作するため、桁違いに低コストで開発できます。

本文書では、Tarosの開発に必要な費用、期間、各段階で何が得られるかをまとめます。

---

## 開発の全体像

開発は4つのPhaseに分かれ、各Phase終了時にGo/No-Go判定を行います。
判定結果が悪ければそこで止められるため、全額をリスクに晒す必要はありません。

| Phase | 期間 | 費用 | 累計 | ゴール |
|-------|------|------|------|--------|
| Phase -1: 原理検証 | 12ヶ月 | 約4.6億円 | 約4.6億円 | 設計の成立/不成立を実験で確定 (8→12ヶ月延長確定) |
| Phase 0: 試作機 | 12-18ヶ月 | 約1.8億円 | 約6.4億円 | 世界初のQEC break-even達成 |

## 各Phaseの詳細

### Phase -1（12ヶ月間）: 原理検証 — 約4.6億円

12ヶ月（GKP実験10ヶ月のクリティカルパス + T10/T11の8ヶ月超過）。
予算約4.6億円（タスク約1.7億円 + 人件費約2.9億円）。

このPhaseの目的は「本当に作れるのか？」に最終回答を出すことです。

**やること:**

- PPLN導波路OPAで12dB以上のスクイージング光を生成する実験
- GPUシミュレーションで誤り訂正の閾値を精密検証
- GKP量子状態の光学的生成実験（世界初クラス）
- 位相ロック制御系の設計と基礎実験

**費用の内訳:**

- チーム11名の人件費（12ヶ月）: 約2.9億円（約2,400万円/月。
  詳細は `analysis/phase-minus1-execution.md` §5.1）
- タスク投資（装置・設備・委託）: 約1.7億円
  （新規約9,150万円 + 既存設備活用約8,250万円。詳細は `analysis/phase-minus1-execution.md` 参照）
- 合計: **約4.6億円**（約2.9億円 + 約1.7億円）

**得られるもの:**

- 「作れる」or「作れない」の明確な判定データ
- 作れる場合: 投資家・パートナー向けの実験実証データ
- 作れない場合: 約4.6億円でプロジェクト終了（被害限定）

### Phase 0（Phase -1完了後12-18ヶ月）: 試作機 — 約1.8億円

Phase -1でGoが出た場合に進みます。

**やること:**

- Taros Edu（教育・研究用モデル）のプロトタイプ製作
- 量子誤り訂正のbreak-even（論理エラー < 物理エラー）を実証
- 100MHzのTDMクロックで安定動作を確認

**得られるもの:**

- 動作する量子コンピュータのプロトタイプ（デスクに載るサイズ）
- 学術論文として発表可能な世界初の成果
- 製品化への技術的見通し

## なぜこの金額で可能なのか

量子コンピュータの開発に通常数百億円かかる理由と、Tarosが桁違いに安い理由を説明します。

### 超伝導方式（IBM/Google）が高い理由

| 必要なもの | 費用 | Tarosでは |
|-----------|------|-----------|
| 希釈冷凍機（15mK） | 約4.5-7.5億円/台 | **不要**（室温動作） |
| クリーンルーム fabrication | 約75-150億円 | **不要**（ファイバ光学系） |
| 超伝導配線・制御装置 | 約15-75億円 | **不要** |
| 大規模冷却設備 | 約7.5-15億円 | **不要** |

### Tarosが安い理由

1. **冷凍機が不要**: 1550nm光子は室温で熱雑音ゼロ（n_th = 3x10^-14）
2. **光ファイバで構成**: 半導体fab不要。既製品を接続するだけ
3. **少人数で実現可能**: 設計の複雑度が低い（部品数が少ない）
4. **NTTの既存技術を活用**: PPLN導波路OPAは共同研究で入手可能

## リスクと段階的判定

全額を一度に投じるのではなく、各段階で成否を判定します。

### 最小投資での判定

| 投資額 | 得られる情報 | 期間 |
|--------|------------|------|
| **75万円** | 設計が原理的に成立するか（GPUシミュレーション） | 2週間 |
| **約870万円（装置直接費。人件費込み完全コスト: 約2,970万円）** | GKP光学実験：スクイージング光が出るか | 5ヶ月 |
| **約1,425万円** | 投資家向けデモ可能なデータ一式 | 5ヶ月 |
| **約5,550万円** | QEC break-even実証（世界初） | 12ヶ月 |

**75万円で「やる価値があるか」がわかり、約870万円で「光が出るか」が確認でき、約5,550万円で「世界初の成果」が出ます。**

**Phase -1 費用定義の明確化（3段階表記）**:

- **75万円（最小判定コスト）**: Stimシミュレーションで設計原理の成立/不成立を判定
- **約870万円（装置直接費）**: GKP実験装置の直接費用（G-EXP1用光学部品+計測器）
- **約2,970万円（完全コスト）**: 約870万円装置費 + 人件費2名x5ヶ月込みGKP実験の完全コスト
  修正タイムライン(5ヶ月)基準。
- 約4,785万円: CV固有技術検証の全装置費用（約870万円含む）+ 基盤設備4,200万円
- 約4.6億円: 人件費込みPhase -1全体予算
  （11名x12ヶ月 約2.9億円 + タスク投資約1.7億円）
- 投資家向けには約4.6億円が正確なPhase -1総額。
  75万円→約870万円→約2,970万円の3段階でリスクを段階的に評価可能。

※GKP実験費用の定義:
装置BOM 約630万円 (OPA x2正確計上) / 装置直接費 約870万円 (予備・消耗品込み) /
完全コスト 約2,970万円 (人件費2名x5ヶ月込み、修正タイムライン基準)

### 失敗した場合の損失限定

| 失敗ポイント | その時点の投資 | 理由 |
|-------------|--------------|------|
| Stimシミュレーションで閾値不足 | 75万円 | 理論的に不成立 → 即中止 |
| OPAスクイージング不足 | 約870万円（装置直接費。人件費込み完全コスト: 約2,970万円） | NTT品質問題 → 改良待ち |
| GKP生成不可 | 約1,425万円 | 物理的制約 → 方式変更 |
| TDM不安定 | 約5,550万円 | 位相ロック問題 → 技術改良 |

## 競合との比較

| 会社 | 累計調達額 | 現在の状態 |
|------|-----------|-----------|
| PsiQuantum | 約1,050億円超 | 未実証、fab建設中 |
| Xanadu | 約375億円超 | Borealis（QECなし） |
| IonQ | 約900億円超 | 32 qubit、非FTQC |
| Google | 数千億円規模 | 105 qubit、break-even実証 |
| **Taros（計画）** | **約8.7億円** | **QEC break-even → 量産** |

Tarosの戦略は「小さく始めて、実証してから拡大する」です。
約8.7億円で競合が数百億円かけても達成できていない成果（デスクサイドFTQC）を狙います。
