# セクション6: ハードウェアサブシステム詳細仕様

> **v4.0注記**: 本セクションの仕様はPhase 0（研究実証機、8ラック+2クライオ構成）のものである。Phase 1/2のコンポーネント数については3.2A節およびセクション2.5を参照。

本セクションでは8ラック構成の各サブシステムについて、部品レベルの仕様・インタフェース・環境条件を規定する。v2.0からの主要変更は以下の通り:

- 光源を1550nm InP QD → 925nm GaAs QDに変更（不区別性・抽出効率の実証値に基づく）
- EO制御をBaTiO₃ → LNOIに変更（商用MPW利用可能性を優先）
- 遅延方式を80段ファイバループ → HCF 600mに変更（損失・パウリフレーム追跡の整合性）
- 検出器をSNSPD 256個 → 8,192個に拡張（行列読み出しで配線数を抑制）

## 6.1 Rack 1: 925nm QD光子源 + LNOI QFC

### 6.1.1 QD単一光子源モジュール

v2.0では1550nm InP QDを採用していたが、Critical Reviewにおいて以下が指摘された:
- InP QDの不区別性（HOM visibility）が実証レベルで93-95%に留まること [Nawrath+ 2023, Appl. Phys. Lett. 122, 244001]
- GaAs QDでは HOM > 98% が複数グループで再現されていること [Tomm+ 2021, Nature Nanotech. 16, 399; Zhai+ 2022, Nature Nanotech. 17, 829]

これを受け、v3.0では925nm GaAs QDを採用し、1550nmテレコム帯への変換にはLNOI量子周波数変換（QFC）を用いる。

**表6.1: QD単一光子源仕様**

| パラメータ | 仕様値 | 根拠・出典 |
|---|---|---|
| 材料系 | InAs/GaAs self-assembled QD | Tomm+ 2021, Nature Nanotech. 16, 399 |
| 発光波長 | 925 ± 2 nm | GaAs QD典型値; Zhai+ 2022 |
| 励起方式 | 共鳴二光子励起（TPE）またはLAフォノンアシスト | TPE: Muller+ 2014; LA: Thomas+ 2021, Phys. Rev. Lett. 126, 233601 |
| 繰り返し周波数 | 50 MHz / QD | サイクル時間2μsに対して十分なマージン（20ns間隔） |
| QD個数 | 12個（アクティブ10 + スペア2） | 光子供給レート 600MHz = 12 x 50MHz |
| g²(0) | < 0.005 | Tomm+ 2021 実証値 0.003 |
| HOM visibility | > 98% （連続光子間） | Zhai+ 2022 実証値 98.5%; Tomm+ 2021 実証値 99.4%（2光子干渉） |
| 抽出効率 | > 80% (first lens) | Open cavity設計; Tomm+ 2021 実証値 57%; Wang+ 2019, Nature Photon. 13, 770 実証値 66%; 目標値はブルズアイ構造による改善見込み |
| end-to-end効率 | > 57% (QD→ファイバ) | 抽出80% × 結合効率90% × 光学系透過80% = 57.6% |
| 動作温度 | 4K（クライオスタット内） | 標準的なQD動作温度; attocube製クローズドサイクルクライオスタット |
| 波長チューニング | シュタルク効果（DC電場）± 3nm | Zhai+ 2022; 電場 0-200 kV/cm |
| 偏光純度 | > 99.5% | クロスダイポール励起 + 偏光フィルタ |
| ジッタ（光子到着時間） | < 30 ps | 典型値 20-25 ps; Schweickert+ 2018, Appl. Phys. Lett. 112, 093106 |
| 寿命（T₁） | ~200 ps | InAs/GaAs QD in open cavity; Purcellファクター ~4 |
| コヒーレンス時間（T₂） | > 350 ps（T₂ ≈ 2T₁ に近い） | フーリエ変換限界に近い; Zhai+ 2022 |

**設計詳細**:

- **クライオスタット**: 各QDは個別の4K closed-cycle cryostat（attocube attoDRY800 相当）に搭載。12個のQDに対し、QDごとに独立した温度・電場制御を行い、波長マッチングを実現。ただし、Phase 2以降ではQD 4個を1クライオスタットに集約（3台構成）を検討。
- **励起レーザ**: 925nm QDに対しTPE励起の場合、462.5nm付近のパルスレーザ（繰り返し50MHz, パルス幅 < 5 ps）。各QD個別に励起パワーをフィードバック制御（π-pulse条件の維持）。
- **ファイバ結合**: PM-SMF（偏波保持単一モードファイバ）で出力。QD→ファイバ結合にはaspheric lens + XYZピエゾステージ（attocube ANPx/z シリーズ）を使用。
- **安定化**: QD波長ドリフトに対し、弱い参照光との干渉によりフィードバック（100Hz帯域）。長期ドリフト（数時間スケール）はDC電場で補正。

### 6.1.2 LNOI QFC モジュール

925nm → 1550nm への量子周波数変換（QFC）は、差周波発生（DFG）を用いる。

**表6.2: LNOI QFC仕様**

| パラメータ | 仕様値 | 根拠・出典 |
|---|---|---|
| 変換プロセス | DFG: 925nm(signal) + 2,290nm(pump) → 1,550nm(idler) | エネルギー保存: 1/925 - 1/2290 = 1/1550 |
| 非線形媒質 | PPLN導波路（LNOI上） | 商用品: HC Photonics, NTT-AT; LNOI版: HyperLight相当 |
| 変換効率 | 87% (内部) / 78% (ファイバ-to-ファイバ) | Zaske+ 2012, Phys. Rev. Lett. 109, 147404（バルクPPLN 75%）; LNOI導波路では閉じ込め強化で改善; Krutyanskiy+ 2023 実証値 57%（イオン→テレコム）; 87%は近年の最良値ベース [Sund+ 2023, Optica 10, 1600 相当] |
| ポンプ波長 | 2,290 nm | OPO出力 or Cr:ZnSe レーザ; IPG Photonics CLT-2300 相当 |
| ポンプパワー | ~100 mW（導波路内） | 変換効率87%達成に必要なパワー; PPLN導波路長 ~3cm |
| g²(0) 劣化 | < 0.005（追加分） | ポンプ誘起ノイズ（SRS/SPDC）を3段VBGフィルタで抑制; Weber+ 2019 |
| ノイズフィルタリング | 3段VBGフィルタ（OptiGrate製相当） + BPF 0.5nm | VBG各段の消光比 > 30dB → 合計 > 90dB; 帯域外ノイズ < 100 cps |
| 偏光保持 | Type-0位相整合（e→e→e）+ 入力偏光制御 | PPLN Type-0は偏光保持が本質的に保証される |
| 動作温度 | 室温（25 ± 0.01°C, TEC制御） | PPLN位相整合温度の安定化 |
| 挿入損失（変換以外） | < 1.0 dB（結合損失含む） | 入力結合0.3dB + 出力結合0.3dB + フィルタ0.3dB |
| 帯域幅 | > 1 GHz（QD光子のスペクトル幅に対して十分） | PPLN導波路長3cmの場合、位相整合帯域 ~50 GHz |

**設計詳細**:

- **ポンプ系**: 2,290nmポンプレーザはCW動作。強度雑音がQFC後の光子品質に影響するため、RIN < -150 dB/Hz を要求。パワー安定化はフォトディテクタ・フィードバックで実現（帯域10kHz）。
- **温度制御**: PPLN導波路の位相整合温度は ~50°C 付近（導波路設計依存）。TEC + サーミスタで ±0.01°C に制御。
- **配置**: 12チャネル分のQFCモジュールを1Uラックマウント筐体に集約（QFCモジュール1個あたり ~50mm x 20mm x 10mm）。ポンプレーザは共有可能（1台の2,290nmレーザからスプリッタで分配）だが、各チャネルのパワー個別調整が必要。
- **信号対雑音比**: QFC後のSNRは >20dB を確保。主要ノイズ源はポンプ光のSPDC（自発パラメトリック下方変換）によるバックグラウンド光子であり、3段VBGフィルタ + 狭帯域BPF（0.5nm）で抑制。

### 6.1.3 ポンプ・励起系まとめ

**表6.3: Rack 1 レーザ系一覧**

| レーザ | 用途 | 波長 | パワー | 台数 |
|---|---|---|---|---|
| TPE励起パルスレーザ | QD励起 | 462.5 nm | 各QDに ~1 mW (ピーク) | 12台（個別制御）or 1台 + 12分岐 |
| QFCポンプCWレーザ | DFG | 2,290 nm | 100 mW x 12ch = 1.2 W (total) | 1-2台 + スプリッタ |
| 参照CWレーザ | QD波長安定化 | 925 nm（可変） | < 1 μW / QD | 1台（共有） |

**Rack 1 消費電力**: ~2 kW（クライオスタット12台: ~1.2kW, レーザ系: ~0.5kW, 制御電子回路: ~0.3kW）
**Rack 1 設置面積**: 標準19インチラック 2本（42U x 2）
**Rack 1 冷却**: 水冷チラー（クライオスタット用）+ 空冷（レーザ・電子回路）
