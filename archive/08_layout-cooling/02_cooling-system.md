## 10.5 冷却系設計（v4.0根本変更）

### 10.5.1 設計方針

> **v4.0の根本認識**: フォトニック量子コンピュータにおいてmK級冷却は必須ではない。SNSPDの動作温度は2-4Kで十分であり、希釈冷凍機（dilution refrigerator）はオーバースペックである。

超伝導量子コンピュータ（IBM、Google等）ではトランスモン量子ビットの動作に10-20mKが必要であり、希釈冷凍機が不可欠である。一方、フォトニック量子コンピュータでは:

- **量子ビット（光子）**: 室温で動作。冷却不要。
- **光学回路（Si₃N₄ PIC、LNOI EO）**: 室温で動作。冷却不要。
- **SNSPD**: 2-4Kで十分な検出効率（>95%）を達成。mK冷却は不要。
- **読み出し回路**: 4Kステージで動作可能。

この認識に基づき、Phase 1以降では希釈冷凍機をパルスチューブ（PT）冷凍機に置き換え、大幅な小型化・省電力化・コスト削減を実現する。

### 10.5.3 Phase 0: BlueFors LD-400×2（v3.0既存、研究実証用）

Phase 0では、v3.0で設計された希釈冷凍機構成をそのまま使用する。

- **機種**: BlueFors LD-400（または同等品）×2台
- **ベース温度**: <10mK（MXCプレート）
- **用途**: SNSPD 8,192個の冷却（4,096個/台）
- **設計根拠**: Phase 0は研究実証機であり、将来のTES検出器評価やmK級実験にも対応可能な汎用性を重視

> **v4.0注記**: Phase 0での希釈冷凍機使用は、SNSPDの動作に必要なわけではなく、研究実証機としての汎用性と、Phase 0で得られる冷却系知見の蓄積を目的とする。

### 10.5.4 Phase 1: PT冷凍機（SHI RDK-415E）+サイドカー

Phase 1では希釈冷凍機を廃止し、PT冷凍機に移行する。

**PT冷凍機仕様**:

| パラメータ | 値 | 根拠 |
|---|---|---|
| 機種 | SHI (Sumitomo Heavy Industries) RDK-415E | 商用PT冷凍機の標準品 |
| 到達温度 | 2.5K（2ndステージ） | SNSPD動作に十分 |
| 冷却能力 | 1.5W @ 4.2K | SNSPD 3,000-3,500個（専門家レビュー修正。erasure-aware符号で半減）+ 配線熱負荷。周波数多重読み出し（MKID方式）により配線熱��荷65mWに削減。**PT冷凍機1台で十分冷却可能** |
| 消費電力 | 3.5kW（コンプレッサ込み） | LD-400の1/6 |
| 重量 | 75kg（コールドヘッド+コンプレッサ） | LD-400の1/9 |
| 振動 | ~10μm p-p（2ndステージ） | **v4.0実戦チーム修正**: 実測値は<5μmではなく~10μm p-p。SNSPDファイバ結合許容差~1μmを大幅に超過。対策必須（下記参照） |
| 冷却時間 | ~4時間（室温→2.5K） | LD-400の1/6 |
| メンテナンス | 20,000時間ごとのコンプレッサオーバーホール | 約2.3年間隔 |

**SNSPD動作温度の妥当性**:
- NbTiN SNSPDの臨界温度: T_c ≈ 10-14K
- 動作温度2.5Kでの検出効率: >95% [Reddy+ 2020での温度依存性データ]
- 動作温度0.8Kでの検出効率: >98%
- 検出効率の差（3%ポイント）はerasure率の微増として符号で吸収可能

**PT冷凍機の振動問題（v4.0実戦チーム追記）**:

PT冷凍機の振動（10μm p-p at 2nd stage）がSNSPDのファイバ結合アライメント（許容差~1μm）を破壊する問題が判明した。これはPhase 1の最も深刻な工学的課題の一つである。

| 項目 | 詳細 |
|---|---|
| **問題** | PT冷凍機の2ndステージ振動が~10μm p-p。SNSPDファイバ結合の許容差~1μmを10倍超過 |
| **対策1** | バッファガス方式による機械的デカップリング（振動を1μm以下に低減） |
| **対策2** | フレキシブルカップリング（ステンレスベローズ等）による振動絶縁 |
| **追加コスト** | $50-80K |
| **追加体積** | +20%（振動絶縁構造分） |
| **検証時期** | Phase 0でのPT冷凍機テスト時に振動対策の有効性を実証 |

### 10.5.5 Phase 2: PT冷凍機一体型

Phase 2ではPT冷凍機をさらに小型化し、検出系と一体化する。

| パラメータ | 値 | 根拠 |
|---|---|---|

| 到達温度 | 2.5K | Phase 1と同等 |
| 消費電力 | 2.5kW | コンプレッサの小型化 |

| サイズ | 400mm×400mm×800mm | デスクサイドユニットとして設計 |

### 10.5.5A 周波数多重読み出しによる冷却系への影響（v4.0 世界的専門家レビュー結果）

Berggren（MIT）提案の周波数多重読み出し（MKID方式）への移行により、冷却系の熱負荷が大幅に削減される。

| 項目 | 行列読み出し（従来） | MKID方式（変更後） |
|------|-------------------|-------------------|
| 4Kステージ配線熱負荷 | 1.2W | **65mW** |
| 必要冷却能力 | 1.5W以上（複数PT冷凍機の可能性） | **0.5W以下（PT冷凍機1台で十分）** |
| 配線本数（4K→室温） | 164本以上 | **<10本** |

この変更により、Phase 1でのPT冷凍機（SHI RDK-415E）1台運用が熱負荷の観点からも確実に実現可能となり、冷却系のサイズ・コスト・消費電力のさらなる削減が可能である。

### 10.5.6 冷却系移行のリスクと対策

| リスク | 確率 | 影響 | 対策 |
|---|---|---|---|
| 2.5KでのSNSPD検出効率が95%未満 | 15% | erasure率増加→符号距離拡大が必要 | Phase 0で温度依存性を実測。不足の場合はHe-3冷凍機（1.5K到達）に変更 |
| PT冷凍機の振動がSNSPDファイバ結合に影響 | 20% | 検出効率の変動 | 振動絶縁マウントの追加。Phase 0のPTテストで検証 |
| PT冷凍機のメンテナンスダウンタイム | 10% | 年間稼働率低下 | 予備コールドヘッドの常備。メンテナンス中はPhase 0の希釈冷凍機で代替運用 |
