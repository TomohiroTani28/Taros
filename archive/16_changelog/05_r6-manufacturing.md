### 1.2E v4.0 R6（第6ラウンド専門家会議）での主要変更概要

第6ラウンドは「量産化・製造可能性・QFC改善」に焦点を当て、実験室スケールから製品スケールへの移行に必要な製造技術と品質管理を追加した。14合意事項（C43-C56）:

**QFC・光源系**:

1. **QFC除去パスのPhase 2オプション化**: InP QD 1550nm直接発光をPhase 2候補として正式評価。Go/No-Go基準: V>0.95+g²(0)<0.02+輝度>0.40（C43）
2. **QFC効率目標を0.57→0.75/0.82に上方修正**: C2N最新プロセス（長周期リッジ導波路）で82%達成。12ch並列検証を追加（C44）
3. **On-chip QFC統合**: LNOI+SiN PIC上にQFCを完全統合。チャネル間アイソレーション>45dB（C45）

**検出系量産化**:

4. **SNSPD Mist CVD+KrFステッパ量産プロセス**: 膜厚均一性±1%、歩留まり93%（64素子）→82%（1,500素子予測）。コスト$200→$30/素子（C47）
5. **SNSPD BIST（Built-In Self-Test）機能**: オンチップテスト光源で各SNSPDの健全性を自動チェック（C48）
6. **冗長度2倍設計（3,000素子）**: SNSPD量産コスト低減により冗長度増加が経済的に実現可能に（C47）

**符号・デコーダ**:

7. **GB [[72,12,6]] をPhase 0-1最小構成用に追加**: 72物理qubitで12論理qubit。保証成果物を4-8→4-12に改善（C49）
8. **Single-Shot/Sliding Windowハイブリッド復号**: 通常時<10ns（Ambiguity Clustering）、バースト時~1μs（Sliding Window）（C50）
9. **リアルタイムノイズ学習エンジン（RTNLE）**: シンドローム統計からベイズ推定でノイズパラメータを連続更新（C53）

**製造・パッケージング**:

10. **クライオパッケージング標準設計**: V-groove光接続+NbTiフレックスRF配線+2段熱アンカー（C46）
11. **Transfer Printingダイアタッチ自動化**: ロボットアーム化で300 QD/4時間。手動比98%時間削減（C56）

**遅延線・インフラ**:

12. **3層遅延構成**: 外部HCF（メイン1μs）+オンチップSiN（短100ns）+SNCM（超短10ns）で外部ファイバ接続点50→10本（C51）
13. **クライオHCFスプール**: Phase 2検討項目としてHCFスプールのクライオスタット内設置を評価（C52）

**ソフトウェア・品質管理**:

14. **量子ベンチマーク標準（QV/CLOPS/LER/TPS/MTBF/FER）+ 自動較正パイプライン（ACP 3層）**: 投資家・ユーザー向けベンチマークとML較正（C54, C55）

### 1.3 主要スペック対比表

| パラメータ | v2.0 | v3.0 (= Phase 0) | 変更根拠 |
|---|---|---|---|
| **光源波長** | 1550nm InP QD | 925nm GaAs QD + LNOI QFC→1550nm | InP QD単一光子源は2026年時点で不定性大。GaAs QDは複数グループで実証済み [Senellart+ 2017, Tomm+ 2021] |
| **検出器** | SNSPD 256個 + 波長多重 | SNSPD 8,192個 行列読み出し（時間多重深さ4） | 波長多重SNSPDは原理的に不可能（超伝導体に波長選択性なし）。行列読み出しは [Wollman+ 2019] で実証 |
| **EO変調器** | BaTiO₃ | LNOI（ニオブ酸リチウム薄膜） | BaTiO₃は研究段階。LNOIは商用製品あり [HyperLight, Photon Forge] |
| **遅延方式** | 80段ファイバループ | ホローコアファイバ600m + パウリフレーム追跡 | 80段ループは損失98.7%（物理的に使用不能）。HCF 600mで損失0.2dB [NKT Photonics] |
| **遅延時間** | 20μs（サイクル） | 2μs（パウリフレーム追跡） | パウリフレーム追跡によりフィードバック不要。遅延は空間的伝搬のみ |
| **符号** | SHYPS一択 | 3パス並行: SHYPS / 4D HPC / 表面符号 | 単一符号依存はリスク大。段階的フォールバック戦略を採用 |
| **閾値計算** | 乗算合成 | Stimフレームワーク直接シミュレーション | 乗算合成は独立ノイズ仮定が不正当 [Gidney+ 2021] |
| **損失閾値** | 2.7%（ベース） | 6%（boosted fusion採用） | Boosted fusion [Bartolucci+ 2023] により閾値を2倍以上改善 |
| **Boosted fusion成功率** | 未採用（fusion成功率75%） | 採用（fusion成功率97%） | 補助ベル対注入により成功率を大幅改善 |
| **物理誤り率** | 不統一（各所で異なる値） | 階層的定義: p_erasure≈40-45%, p_depol≈0.7-1.0% | FBQC固有のエラー構造を正確に反映 |
| **光モード数** | 3,200 | 再計算→約12,000（空間多重主体） | 検出器数・符号要件から再導出 |
| **コスト** | 27-33億円（矛盾あり） | 70-90億円（段階的投資） | SNSPD 8,192個、QFC追加等を反映。Phase 1: 25億 / Phase 2: 30億 / Phase 3: 15-35億 |
| **論理量子ビット目標** | 100（Phase 3） | 100（Phase 3）[変更なし] | — |
| **クライオ冷却** | JTクライオクーラ 4K | JTクライオクーラ 4K [変更なし] | — |

### 1.4 達成目標

> **v4.0 Phase体系注記**: v3.0で定義されていたPhase 1-3は、v4.0ではPhase 0内のサブフェーズ（Phase 0a/0b/0c）として再編される。v4.0のPhase 1/2は小型化・実用化を目標とする新規Phaseである。

- **Phase 0a**（Year 1-2, 旧Phase 1）: 単一リソースステート生成・検証、boosted fusion実証、SNSPD行列読み出し検証。論理量子ビット0個。研究実証機構成（8ラック+2クライオ、60m²）。
- **Phase 0b**（Year 2-4, 旧Phase 2）: 論理量子ビット4-10個、QEC実証、Stimベース閾値の実機検証。
- **Phase 0c**（Year 4-6, 旧Phase 3）: 論理量子ビット100個、アプリケーション実行。
- **Phase 1**（2029-2030）: 小型実用機。2ラック+サイドカー構成、床面積6m²、消費電力8kW、16論理qubit。PT冷凍機への移行。
- **Phase 2**（2031-2032）: 次世代小型機。1ラック+外部冷却構成、床面積4m²、消費電力6kW、48論理qubit。Quantum LDPC符号への移行。
