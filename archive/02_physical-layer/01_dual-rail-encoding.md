## セクション4: 物理層 — DVデュアルレール符号化と925nm QD + QFC光源

### 4.1 デュアルレール符号化

#### 4.1.1 基本定義

デュアルレール符号化では、1量子ビットを2つの光学モード（空間モードまたは偏光モード）の単一光子占有で表現する。

```
|0⟩_L = |1,0⟩ = a₁†|vac⟩    （モード1に光子1個、モード2に光子0個）
|1⟩_L = |0,1⟩ = a₂†|vac⟩    （モード1に光子0個、モード2に光子1個）
```

光子損失は消失エラー（erasure error）として検出可能:
```
|ψ⟩_L = α|1,0⟩ + β|0,1⟩  → 損失 →  |0,0⟩（検出可能: 両モードとも光子なし）
```

この「損失 = 検出可能な消失」がデュアルレール符号化の最大の利点であり、FBQC全体のエラー訂正戦略の基盤である。

#### 4.1.2 Dephasing error管理（v3.0で追加）

v2.0では省略されていたデフェージングエラーの管理を追加する。

デュアルレール量子ビットのデフェージングは、2モード間の相対位相の揺らぎとして発生する:

```
|ψ⟩ = α|1,0⟩ + β e^{iφ}|0,1⟩   （φ: 位相揺らぎ）
```

位相揺らぎの発生源と対策:

| 発生源 | 大きさ | 対策 |
|---|---|---|
| ファイバ経路長差 | δφ ∼ 2π × (δL/λ) | 能動位相安定化（PZT + PID, <λ/100残留誤差） |
| 温度変動 | dn/dT ≈ 10⁻⁵/K（シリカ） | 温調±0.01K, HCF使用（dn/dT ≈ 0） |
| EO変調器残留位相 | δφ ≈ 0.01 rad | キャリブレーション（1分間隔） |
| QFC位相ノイズ | δφ ≈ 0.005 rad | ポンプ光安定化（RIN < -160 dBc/Hz） |

デフェージングに起因する誤り率:

```
p_dephasing ≈ (δφ_total)² / 2 ≈ (0.015)² / 2 ≈ 1.1 × 10⁻⁴
```

これは p_depolarizing ≈ 0.7-1.0% に対して十分小さく、全体の誤りバジェットに対する影響は限定的である。
