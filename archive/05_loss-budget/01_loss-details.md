# セクション7: 損失予算（現状 vs 目標）

## 7.1 損失予算詳細（全面改訂）

v2.0の損失予算はdB加算の近似を用いていたが、v3.0では厳密な乗法的計算（生存確率 P_survive = Π(1 - ε_i)）を採用する。

### 7.1.1 損失経路の定義

光子が通過する全段階を以下に定義する:

```
QD発光 → QD抽出 → ファイバ結合 → QFC → MUX → PICチップ入力
→ PIC内（fusion gate） → PICチップ出力 → HCF遅延（蒸留レーンのみ）
→ ファイバ→SNSPD結合 → SNSPD検出
```
