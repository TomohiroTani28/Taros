| コンポーネント | 数量 | コスト |
|-------------|------|--------|
| QD光源 | 24個配置、6個稼働 | $350K |
| SNSPD | 18個（個別読み出し） | $150K |
| LNOI PICチップ | 1枚 | $200K |
| SiN遅延線チップ | 1枚 | $50K |
| PT冷凍機（4K） | 1台 | $200K |
| FPGA + 古典制御 | 1式 | $80K |
| 光学系・ファイバー | 1式 | $100K |
| 合計 | — | **$1.2M** |

サイズ: 光学テーブル2台 + クライオスタット1台 = **3m²**
HCF遅延線: **使用しない**（SiNオンチップ遅延線のみ）
