## 16.2 コスト削減ロードマップ

### 16.2.1 段階的コスト削減計画

v3.0の基準コスト70-90億円から、技術成熟に伴い段階的にコストを削減する計画を策定する。

| 時期 | 目標コスト | 主要削減策 | 削減メカニズム |
|---|---|---|---|
| **短期（2026-2028）** | 25億円 | SNSPD-on-chip化、MOCVD QD量産、MLデコーダ符号距離低減 | SNSPDコストを1/10に削減（チップ集積による量産効果）。MOCVD成長によるQDの量産化でQDコストを1/5に削減。MLデコーダの閾値改善により必要符号距離を低減し、物理量子ビット数を削減 |
| **中期（2029-2031）** | 10億円 | 量子LDPC符号導入 | 物理量子ビット数を1/10に削減（SHYPS比）。検出器数・遅延線数も比例して削減 |
| **長期（2032-）** | 5億円以下 | 量子光チップ（モノリシック集積） | QD + 干渉計 + 検出器をSi基板上にモノリシック集積。半導体ファウンドリの量産プロセスを適用 |

### 16.2.2 Phase 0最小構成

技術検証のための最小構成として、約4,500万円のPhase 0システムを以下の通り設計する。

| コンポーネント | 数量 | コスト（万円） |
|---|---|---|
| 925nm GaAs QD | 4個 | 500 |
| LNOI QFC | 4個 | 200 |
| Si3N4 PLC | 1個 | 300 |
| SNSPD | 32個 | 1,600 |
| クライオスタット | 1台 | 800 |
| 制御系（FPGA + PC） | 1式 | 600 |
| 光学系（ファイバ、レンズ等） | 1式 | 500 |
| **合計** | | **4,500** |

この最小構成での情報投資としてのROIは340%と見積もられる（Phase 0で得られる技術知見の経済価値から算出）。
