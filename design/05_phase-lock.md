## 1. CV方式における位相安定性の重要性

CV量子計算では、スクイーズド光の位相がスクイージング品質を直接決定する。

| 位相誤差 Δφ (実効高周波) | 劣化 (V_anti=20 SNU, 13dB anti-squeezed) | 影響 |
|------------|------------------|------|
| ±0.05° (PLL BW≥500kHz) | **<0.002dB** | 事実上無視可能 |
| ±0.15° (PLL BW~100kHz) | ~0.01dB | 無視可能 |
| ±0.5° (PLL BW~10kHz) | ~0.17dB | 許容範囲（旧: 0.55-0.85dBから大幅緩和） |
| ±1° (no PLL) | ~0.6dB | σ_eff=10.8dB→~10.2dB、要改善 |
| ±3° | ~5dB | スクイージング消滅 |

**設計要件**:
- 全光路でRSS位相安定性 ±0.5° 以内
- **PLL帯域 ≥ 500kHz** (反スクイージング~13dBとの混入を抑制)
- 実効高周波位相ノイズ: φ_hf = RSS × √(f_PLL/f_Nyquist) ≤ 0.05°

**物理的根拠**: σ_anti²×sin²(φ_hf)がσ_sqz²に対して1%以下であること。
σ_anti²/σ_sqz² = 10^((13+13)/10) ≈ **400** (単一パスOPA反sq=13dB。旧値1260は18dB反sq誤用) → sin(φ_hf) < 0.01/√400 = 5.0×10⁻⁴ → φ_hf < 0.029° (1%理想条件)。
実設計: φ_hf=0.05°で劣化0.006dB（06_noise-budget §2.2 #10）を許容。

## 3. Layer 1: マスタークロック

### 3.1 超安定CWレーザ

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| 波長 | 1550nm | テレコム帯、OPA信号光と同一 |
| 出力パワー | 100mW | SHG + LO分配に十分 |
| 線幅 | <1kHz (1ms) | 位相安定性要件から |
| RIN | <-165dBc/Hz | スクイージング測定の限界にならない |
| 候補 | NKT Koheras BASIK E15 | 約225万円, 商用品 |
| (代替) | OEwaves OE4040 | 約300万円, 超狭線幅 |

### 3.2 ポンプ光源: 775nm直接LD + SHG位相参照

**変更**: 50mW SHG→光増幅の方式を廃止。775nm直接LDに変更。

| パラメータ | 仕様 |
|-----------|------|
| **主ポンプ光源** | **775nm DFB-LD + TA (2.5W)** |
| 位相参照 | 1550nm master → PPLN SHG → 775nm参照光 (10mW) |
| 位相ロック | 775nm LD出力を SHG参照光に位相ロック（ヘテロダイン検出） |
| 位相関係 | φ_pump = φ_SHG_ref + const = 2×φ_master + const |

**方式の変更理由**:
- 旧方式: 50mW@1550nm → SHG → 5-10mW@775nm → 光増幅500mW。SHG効率問題あり。
- **新方式**: 775nm DFB-LD + TA (テーパードアンプ) で2.5W出力 + SHG参照光で位相ロック。
  - DFB+TA: Eagleyard DFB seed + Toptica BoosTA相当、約120万円。butterfly package可。
  - SHG参照: 10mW程度で十分（位相検出用のみ）。1550nm 100mW → SHG ~5mW@775nm。
  - 2.5W ÷ 8 OPA = 312mW/OPA（1×8スプリッタ損失込みで~250mW/OPA、要件200mWに対し25%マージン）。
| 位相ロック手法 | 仕様 |
|-------------|------|
| ヘテロダイン検出 | 775nm LD出力と SHG参照を合波 → ビート周波数検出 |
| ロック帯域 | 100kHz（LD電流変調で十分） |
| 残留位相差 | < 0.1° (SHG位相は原理的にmaster×2で固定) |

**SHGの位相優位性は維持**: ポンプ(775nm)と信号(1550nm)の位相関係は、SHG参照を介して原理的にロック。bow-tie OPOの共振器位相ロックより圧倒的に簡素。

### 3.3 ポンプパワー分配

| パラメータ | 仕様 |
|-----------|------|

## 4. Layer 2: ポンプ位相ロック

### 4.1 アーキテクチャ

```
775nm DFB-LD + TA (2.5W) ──► 1×8 splitter (~250mW/OPA) ──┬─► OPA#1
                                                           ├─► OPA#2
                                                           ├─► OPA#3
                                                           ├─► OPA#4
                                                           ├─► OPA#5
                                                           ├─► OPA#6
                                                           ├─► OPA#7
                                                           └─► OPA#8
```

### 4.2 位相安定性

**マスターレーザ→SHG→OPAの光路は全てファイバ接続。**

ファイバの位相ドリフト主因:
- 温度変動: ~40 rad/m/℃ (PMF @1550nm, 文献値; §5.4の8100rad/K/200m=40.5rad/m/℃と整合)
- ラック内温度制御: ±0.5℃
- OPAモジュール内ファイバ長: ~1m
- **位相ドリフト**: ~40 rad/m × 1m × 0.5℃ = **~20 rad** (OPAモジュール内)

→ これは激しすぎるので**アクティブ位相ロック必要**。

### 4.3 ポンプ位相ロック方式（2段構成: EOM高速 + PZT低速）

**設計要件PLL BW≥500kHzを達成するため、高速EOM + 低速PZTの2段構成を採用。**

```
1550nm master ──► [EOM phase (1550nm LNOI)] ──► [SHG] ──► 775nm pump ──► [PZT] ──► [OPA] ──► squeezed output
                    ↑               ↑                              │
                    │               │                              ▼
                    │               │                    [tap 1%] ──► [PD]
                    │               │                                  │
                    │               └── [slow PID, 10kHz] ◄────────────┤
                    │                                                   │
                    └────── [fast PID, 1MHz BW] ◄──────────────────────┘
```

**動作原理**:
- **EOM（高速）**: **1550nm LNOI位相EOM（SHG前段に配置）**。帯域>40GHz（商用品: iXblue/Exail LN series）。
  1550nmでの位相変調φがSHGにより2φとして775nmに伝達 → Vπ_eff = Vπ/2 = 1V相当。
  10kHz〜1MHzの位相ノイズを抑制。**775nm EOM不要（商用品問題を完全回避）**。
- **PZT（低速）**: DC〜10kHzのドリフト補正。EOMの中心電圧をゼロ付近に維持（動的レンジ確保）。

| パラメータ | 仕様 |
|-----------|------|
| センサ | OPA出力のタップ(1%)をPDで検出、反スクイージング位相で位相誤差抽出 |
| 高速アクチュエータ | **1550nm LNOI位相EOM (Vπ=2V, BW>40GHz) — SHG前段配置** |
| 低速アクチュエータ | PZTファイバストレッチャ (30μm stroke @775nm) |
| 高速PIDループ帯域 | **1MHz** (要件500kHzの2倍マージン) |
| 低速PIDループ帯域 | 10kHz (ドリフト補正) |
| 残留位相誤差 | <±0.1° (RMS, 高速ループ後) |
| EOMコスト | 約22万円/ch × 8 = 約180万円 |
| PZTコスト | 約15万円/ch × 8 = 約120万円 |
| **合計** | **約360万円** |

**根拠**: PZT単独(BW=10kHz)では10kHz〜500kHz帯の音響/機械ノイズが未抑制。
ファイバ系の位相ノイズスペクトルは1/f特性で10kHz以上にも有意な成分があり、
V_anti=20 SNU(13dB anti-squeezed)との混入で劣化を招く(単一周波数φ=0.5°で~0.08dB@Phase 2+ PIC。BW=100kHz全帯域積分では~0.17dB)。EOM高速ループで実測φ_hf<0.05°を保証。

## 5. Layer 3: LO位相ロック

### 5.1 アーキテクチャ

ホモダイン測定のLO (Local Oscillator) をマスターレーザから分岐。

```
Master laser (1550nm) ──► 1×4 splitter ──┬─► LO for Homodyne A
                                          ├─► LO for Homodyne B
                                          ├─► (spare)
                                          └─► Phase reference (monitor)
```

### 5.2 LO位相制御

測定基底角度θの設定 = LO位相の設定:

```
LO ──► [LNOI EOM] ──► Homodyne detector
            ↑
            │ V_θ (基底角度に応じた電圧)
            │
       [FPGA: 基底選択ロジック]
            ↑
            │ (GKPデコーダからの適応指示)
       [Feedforward chain]
```

| パラメータ | 仕様 |
|-----------|------|
| EOM | LNOI Mach-Zehnder, Vπ = 2V |
| 切替速度 | <1ns (帯域 >40GHz) |
| 位相分解能 | 0.1° (DAC 14bit) |
| 長期安定性 | ±0.5° (DCバイアスドリフト補正) |

### 5.3 Pilot Tone位相監視

全光路の位相ドリフトをリアルタイム監視するため、弱い古典光（Pilot Tone）を注入:

```
Master laser ──[1% tap]──► Pilot Tone (10μW)
                               │
                               ▼
                    全光路（OPA→TDM→ホモダイン）を量子光と並行伝搬
                               │
                               ▼
                    [ホモダイン出力のDC成分を監視]
                               │
                    位相ドリフト = arg(Pilot Tone signal)
                               │
                               ▼
                    [全チャネルの位相補正テーブル更新]
```

| パラメータ | 仕様 |
|-----------|------|
| Pilot Tone強度 | 10μW (量子信号の~1000倍) |
| 周波数シフト | +80MHz (AOMで周波数多重) |
| 監視帯域 | 1kHz (温度ドリフト追従) |
| 位相精度 | ±0.1° |

### 5.4 遅延線位相安定化

長遅延PMF (37-200m, d=3:37m / d=5:102m / d=7:200m) は熱係数~8100 rad/K/200mの位相ドリフトを受ける。
Pilot Tone (1kHz BW) では音響帯域(100Hz-10kHz)をカバーできないため、
**遅延線専用PZTファイバストレッチャ**を追加:

```
PMF長遅延出力 ──► [PZT stretcher (遅延線用)] ──► BS₂
                        ↑
                        │ [PID 10kHz BW]
                        │       ↑
                        │  [Pilot Tone DC出力 → 位相検出]
                        └───────┘
```

| パラメータ | 仕様 |
|-----------|------|
| アクチュエータ | PZTファイバストレッチャ (30μm stroke) |
| 動的レンジ | ~122 rad @1550nm (30μm stroke = 2π×30μm/1.55μm, 温度±0.02Kに対応) |
| PID帯域 | 10kHz (音響振動追従) |
| センサ | Pilot Tone DC出力からの位相抽出 |
| コスト | 約15万円 × 1本 (Max: 約15万円×1, Pro: 約15万円×1, Edu: 不要(20m)) |

**設計保証**: 環境温度±0.5℃をPeltier制御で±0.01℃に抑制(Zone B)、
残留±0.01℃ドリフト = 81 rad → PZTレンジ122 rad内で安全（マージン: 122-81=41 rad, **34%**）。
音響振動(~1 rad RMS @100Hz-10kHz) → PZT 10kHz BWで完全抑制。

**10kHz-500kHz帯ギャ���プ分析**:
Layer 2 EOM PLL(BW=1MHz)は遅延線「前」のOPA光に適用される。遅延線PZT(10kHz BW)との間に
10kHz-500kHz帯のギャップが存在するように見えるが、以下の理由で問題なし:
- PMF遅延線通過中に発生する位相ノイズ源: (a)熱ドリフト(<100Hz)→PZT対応、
  (b)音響振動(100Hz-10kHz)→PZT対応、(c)GAWBS: radial/torsionalモード(>20MHz)→06テーブル#16で0.005dB計上済み
- **axial acoustic基本モード**: v_sound/(2L) = 5000m/s÷(2×200m) = **12.5kHz** がギャップ帯に存在する。
  ただしaxialモードのファイバ位相変調結合係数はradialモードの~10⁻³倍(Shelby+ 1985)であり、
  200m PMFでの寄与は < 10⁻⁶ rad²/Hz × BW(500kHz) ≈ 5×10⁻⁴ rad RMS → ~0.03°。
  RSS 0.41°に対し0.03°追加はRSS合計に<0.1%寄与で無視可能。
- 10kHz-500kHz帯に**有意な**PMF位相ノイズ源は存在しない（axialモードは結合弱で無視可能）
- Phase -1でPMF 200mの位相ノイズスペクトル(1Hz-10MHz)を実測し確認する

### 5.5 PLL ロック喪失復帰プロトコル

| パラメータ | 仕様 |
|-----------|------|
| ロック喪失検出 | Pilot Tone振幅 < -3dB (10μs以内に検出) |
| 復帰シーケンス | EOM高速スキャン(10MHz) → ロック点再捕捉 → PID収束 |
| 復帰時間 | **目標: <21μs (3 QECサイクル)** — Fast Recovery Mode参照 |
| QEC中断処理 | ロック喪失中のモードをburst erasure宣言 → デコーダ処理 |
| MTBF (ロック喪失) | >10⁴ s推定 (=0.01% duty cycle でerasure追加<0.1%) |
| ロック回復失敗時 | 全モードerasure → 1秒後にフルリセット再ロック |

**Fast Recovery Mode**:
通常復帰(50-200μs)では200μs/7μs ≈ 29 QECサイクルのburst erasureが発生し、
d=7表面符号のburst耐性(d/2=3連続ラウンド)を大幅に超過 → 論理エラー確定。

| 対策 | 原理 | 目標復帰時間 |
|------|------|-------------|
| EOM高速位相追従 | lock-loss検出10μs後、EOM(BW>40GHz)のみで粗位相追従開始 | <5μs (EOM capture) |
| PZT fine-lock並行 | EOM追従中にPZT PID loop再収束 | 5-21μs (PZT settle) |
| Burst erasure宣言 | lock-loss～EOM capture完了まで全モードerasure | 最大3 QECサイクル |

EOM dynamic range ±Vπ (=±π rad)で21μs間の位相ドリフト(≈0.2rad@100Hz)をカバー可能。
**目標**: burst erasure ≤ 3 QECサイクル(21μs) → d=7 burst耐性内に収める。

**QEC影響**: Fast Recovery成功時: 3 QECサイクルのburst erasure (d=7で訂正可能)。
通常復帰(フォールバック): ~30サイクル → 論理エラー1回。MTBF>10⁴sの場合、
論理エラーレートへの寄与: 1/(10⁴s × 146kHz QEC cycles) ≈ 7×10⁻¹⁰ (p_L<<10⁻⁶に対し無視可能)。

### 5.6 パルスOPA対応 (Option B)

Option B (パルスポンプOPA) では以下の制御系変更が必要:

- **Layer 2 PLL**: OPA出力タップからの誤差信号が断続的(duty 30%)。サンプル＆ホールド回路をPD後に追加し、パルス到達時にホールド→PID入力。3ns/10ns=30%
- **Pilot Tone**: パルスOPA OFF時はPilot Toneが線形通過し位相基準として常時利用可能。OPA ON時のパラメトリック相互作用はPilot Tone帯域外配置で回避。
- **ホモダイン検出**: ADCサンプリングがパルス中央に同期(14_clock-distribution.md §5)。TIA帯域(500MHz)とパルス幅(≥3ns)の整合条件: 立上り0.7ns×4 < パルス幅。Phase -1で実測確認。
- **S&H回路要件**: ドループレート<1mV/μs、アパーチャジッタ<5ps、取得時間<2ns。候補: AD783 (500MHz BW, 2ns acquire)。PLL BW 1MHzに対しサンプルレート100MHz(Nyquist 50MHz)で十分余裕。

## 6. 全体位相バジェット

| 位相誤差源 | 寄与 (°RMS) | 対策 |
|-----------|------------|------|
| ポンプ位相ロック残差 | 0.1 | EOM高速PLL |
| LO位相設定誤差 | 0.1 | 14bit DAC |
| ファイバ温度ドリフト残差 | 0.3 | Pilot Tone補正 |
| OPA内部位相雑音 | 0.2 | PPLN導波路の本質的安定性 |
| 振動起因 | 0.1 | 光学テーブルなし（ファイバ系） |
| EOM駆動電圧雑音 | 0.1 | 低雑音DAC |
| **RSS合計** | **0.41°** | **要件±0.5°を達成** |

**スクイージング劣化計算**:
- RSS位相誤差: 0.41° (DC〜50MHz帯域)
- PLL帯域: ≥500kHz → 500kHz以下の成分は追従済み
- 測定に影響する高周波位相ノイズ: φ_hf ≈ 0.41° × √(500kHz/50MHz) = **0.041°**
  06_noise-budget §2.2 #10ではこの値に安全マージン+20%を付加し、設計保守値 0.05° で劣化を計算。
  (注: √帯域比スケーリングはホワイトノイズ仮定。実際のファイバ系は1/f特性のため
   PLL BW以上の積分パワーはこの推定より小さい可能性がある。保守値として採用。)
- 反スクイージング: V_anti = 20 SNU (13dB anti-squeezed; V_sqz×V_anti=1, 最小不確定性。旧σ_anti²表記廃止)
- 追加ノイズ: V_anti × sin²(φ_hf) = 20 × (7.2×10⁻⁴)² = 1.0×10⁻⁵ SNU → **<0.001dB (事実上無視可能)**
- (06_noise-budgetでは設計保守値0.05°で1.5×10⁻⁵ SNU→<0.002dBと計上)
- **劣化: <0.002dB** (PLL BW≥500kHz前提。anti-sq=13dBで位相ノイズは設計制約から事実上脱落)

**警告**: PLL BW < 100kHzの場合、φ_hf ≈ 0.35°となり劣化~0.25dB。
Phase -1実験でのPLL帯域実測が必須（05_phase-lock §5 Pilot Tone 1kHz更新では不十分 → 高速PLL実装が必要）。

**PLL BW未達時フォールバック戦略**:
| 実測BW | σ_eff影響 | 対策 |
|--------|----------|------|
| ≥500kHz | 0.006dB (設計値) | 計画通り |
| 300-500kHz | ~0.03dB | 許容範囲内、マージン縮小のみ |
| 100-300kHz | 0.05-0.15dB | d=7維持可能、p_L 2-5倍悪化 |
| <100kHz | 0.25dB+ | σ_eff=11.3dB。対策: (a) EOM帯域拡張、(b) d=9拡張、(c) QD追加(Step B前倒し) |

**BW=100kHz時の定量影響**: BSモデルでは追加劣化~0.17dB。Phase 2+ PIC構成 σ_eff 10.9→10.7dB、p_L悪化~2倍。PLL BW≥500kHz推奨は維持。
BW=50kHz以下ではLevel A未達 → QD追加が必須。
Phase -1 Go基準: PLL BW ≥ 200kHz (Conditional Go), ≥ 500kHz (Full Go)。
