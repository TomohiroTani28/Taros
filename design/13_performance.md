## 1. 性能モデル概要

暫定版から確定版への主要変更:
- bow-tie OPO → PPLN導波路OPA: σ_gen = 12dB → **13dB** (Phase 1/2+共通。旧15dB要求は撤廃)
- **σ_eff公式修正**: 旧式 σ_eff = σ_gen − L(dB) は物理的に不正確。正しくは **V_eff = η×V_sqz + (1−η), σ_eff = −10×log₁₀(V_eff)**（ビームスプリッタモデル）
- **Option B（パルスポンプOPA）を推奨ベースラインに変更**: EO gate 0.3dB損失排除
- フェーズ別実効スクイージング:
  - Phase 1 (離散光学系, 13dB, L=0.39dB): **σ_eff = 8.5dB** (break-even実証) V_non-loss補正
  - Phase 2+ PIC (現実的, 13dB, L=0.27dB): **σ_eff ≈ 9.3dB** (non-loss込み), p_L(d=7, MWPM) ≈ **3.3×10⁻⁴** (製品スペック: p_L≲10⁻³) erfc再計算 + V_non-loss=0.010 SNU補正
  - Phase 2+ PIC (理論限界, 13dB, L=0.15dB): σ_eff = 10.8dB, p_L ≈ 6.1×10⁻⁷ (全条件達成時のみ)
- 旧値 σ_eff = 11.5dB はdB減算による過大評価（廃止）

### 3.1 スケーリング則

Noh & Chamberland (2022) + Stafford-Menicucci-Walshe (2025):

```
p_L(d, σ_eff, N_QD) ≈ A × (p_phys / p_th)^((d+1)/2)

p_phys(σ_eff) = erfc(√π / (4σ)) / 2
  σ = √(V_eff / 2)  [GKP変位ノイズ標準偏差, SNU基準, macronode BS後]
  V_eff = η×V_sqz + (1−η) [ビームスプリッタモデル]
  σ_eff_dB = -10 log₁₀(V_eff) [スクイージングレベルの定義]
  ※ σ = √(0.5 × 10^(-σ_eff_dB/10)) と同値。V_eff = 10^(-σ_eff_dB/10)
  ※ 検算: σ_eff=10.8dB → V_eff=0.0832 → σ=0.2039 → erfc(√π/(4×0.2039))/2 = erfc(2.17)/2 ≈ 1.0×10⁻³ ✓

パラメータ (06_noise-budget.md §4.1準拠):
  A = 0.03 (soft-info MWPM prefactor, Noh & Chamberland 2022)
  p_th = 0.59% (conservative, hard MWPM) / p_th_eff = 1.5% (soft-info MWPM)

  - 上記A, p_thはMWPMデコーダのフィッティング値。Union-Find(UF)使用時:
    - UF prefactor: A_UF ≈ 0.05-0.10 (Delfosse & Nickerson 2021)
```

### 3シナリオ σ_eff導出根拠

各シナリオのσ_eff実効値は以下の式で計算 (**ビームスプリッタモデル**):

**基本式**: V_eff = η × V_sqz + (1−η) + V_Δ + V_RIN
**σ_eff(dB)** = −10 × log₁₀(V_eff)

- η = 10^(−L_opt/10): 全光学透過率
- V_sqz = 10^(−σ_gen/10): 生成スクイーズド分散
- V_Δ ≈ Δ²: 有限エネルギーGKPによる追加分散
- V_RIN ≈ ρ×(d-1)×V_sqz: TDM相関ノイズ分散

> **注記**: 旧式 σ_eff = σ_gen − L_opt − L_Δ − L_RIN (dB減算) は物理的に不正確。
> 損失が大きい場合、dB減算は実効スクイージングを大幅に過大評価する。

楽観 (Phase 2+ PIC): σ_gen=13, L=0.15, Δ=0.08, RIN −155 → σ_eff≈10.5dB
現実 (Phase 2+ PIC): σ_gen=13, L=0.15, Δ=0.12, RIN −150 → σ_eff≈9.8dB
保守 (Phase 1 離散): σ_gen=13, L=0.39, Δ=0.15, RIN −145 → σ_eff≈7.8dB 旧15dB→13dB

### TDM相関ノイズの影響推定

TDM方式では全モードが同一CWポンプから生成されるため、ポンプRINが共通ノイズとして混入する。

- DFB+TA典型RIN: −150 dB/Hz → 隣接モード相関 ρ ≈ 0.03
- p_th低下推定: ~3-5%（Fowler+ 2012 相関ノイズ解析ベース）
- p_L(d=7)への影響: 1.5-2倍増加（6.1×10⁻⁷ → 0.9-1.2×10⁻⁶）(MWPM基準)
- **ポンプRIN仕様**: −150 dB/Hz 以上を規定。Phase -1で実測検証要。
- **劣悪TA時（−140 dB/Hz以下）**: p_th,corr ≤ 0.80%、p_L(d=7) ≈ 2×10⁻⁶（製品要件超過リスク）

### 3.3 改善要因分析

## 8. CV方式ネイティブ応用（キラーアプリ）

### 8.1 分子振動スペクトルシミュレーション（Phase 0, CV唯一）

**出典**: Nature Communications (2024), 大規模フォトニックネットワーク実証

CV方式の**光モードが分子振動モードに自然に対応**するため、分子振動スペクトルを指数的効率で計算可能。DV方式では原理的に非効率（ボソニック→qubit変換のオーバーヘッド）。

- **Tarosの8ch OPA → 8モード分子の振動スペクトル即座にデモ可能**
- ドゥシンスキー回転 = ビームスプリッタ操作（TarosのBS網で直接実装）
- 変位操作 = コヒーレント変位（EOMで実装済み）
- フランク・コンドン因子の量子サンプリング → GBS（ガウシアンボソンサンプリング）で実行

### 8.2 量子重力シミュレーション（Phase 0-1）

CVボソニックモードが量子場の量子に直接対応。詳細は `01_system-architecture.md` Appendix B参照。
- LQGスピンフォーム: Phase 0 (数量子ビット)
- SYK/ワームホール: Phase 1 (10-50論理量子ビット)

### 8.3 金融モンテカルロ（Phase 1+）

GBS（ガウシアンボソンサンプリング）による確率分布サンプリングはCV方式のネイティブ計算。重み付き共分散行列を使用したポートフォリオ最適化やオプション価格付けに量子優位性を示す可能性（§8.14 GBS注意事項参照: 非重み付きグラフGBSは古典的にサンプリング可能）。

### 8.4 コヒーレントイジングマシン（CIM）モード（Phase -1, 追加HWゼロ）

**出典**: Science Advances (NTT 100,000スピンCIM), Nature Comms 2026 (Kerr CIM), Light: Sci & Apps 2026

**TarosハードウェアはCIMとほぼ同一構成**。NTTのCIMは「PPLN導波路 + ファイバリング共振器 @ 1550nm」で構成されており、Tarosの「PPLN OPA + 200m PMF遅延線 @ 1550nm」と物理的に等価。

- **切替方法**: OPAポンプ強度を閾値以下に下げるだけでCIMモードに移行（SW/FPGA変更のみ）
- **容量**: 200m遅延線 → ~2,000パルス（100MHz）のイジングスピンを格納可能
- **応用**: 組合せ最適化（Max-Cut, 独立集合問題, 交通最適化, スケジューリング）
- **FPGAフィードバック**: 既存の測定→FPGA→EOMループをCIMの測定フィードバック結合に転用
- **ビジネス価値**: FTQC到達前（Phase -1/0）で組合せ最適化デモを即座に提供可能

### 8.5 量子リザーバコンピューティング（Phase -1, 追加HWゼロ）

**出典**: Nature Photonics 2026 (Paparelle et al.), Nature Comms 2025 (スケーラブルフォトニックリザーバ)

**Tarosのアーキテクチャそのものがリザーバコンピュータ**。OPA → PMF遅延 → ホモダイン → FPGA → EOM のループ構造は、量子リザーバ計算の標準的アーキテクチャと完全に一致。

- **実装**: SW変更のみ。入力信号をEOMで注入し、ホモダイン出力を読み出し層として利用
- **14bit soft-info**: リザーバの表現力を大幅に向上（DV方式のbinary出力では不可能な高精度）
- **WDM 8ch**: 8並列リザーバチャネルとして同時に異なるタスクを実行可能
- **応用**: 時系列予測、異常検知、カオス信号予測、音声認識
- **フォールトトレランス不要**: 生のスクイーズド光+ホモダイン測定で動作。QEC前の即時デモ

- **CV fault tolerance一般理論** (Nature Comms 2026, Matsuura et al.): 一般的Markovノイズに対するFT閾値の存在を証明
- **Focked-up ZX Calculus** (arXiv:2406.02905): CV量子計算の完全公理化 → TDMコンパイラの理論基盤候補
- **GBS注意事項**: 非重み付きグラフ上のGBSは古典的に多項式時間でサンプリング可能 (Nature Comms 2025)。量子優位性主張は重み付き/構造化グラフ問題に限定すべき
