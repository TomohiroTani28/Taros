## 1. アーキテクチャ概要

### 1.1 方針転換の根拠

設計チームによる調査・議論の結果、**ハイブリッドCV+QD方式を主軸、DV-FBQC方式をフォールバック**とする方針に合意した。

| 評価軸 | DV-FBQC (旧主軸) | CV-TDM (純粋CV) | ハイブリッドCV+QD (新主軸) |
|--------|---|---|---|
| FTQC確実性 | ★★★★ | ★★ | ★★★ |
| デスクトップ適合性 | ★★★ | ★★★★★ | ★★★★ |
| コスト (量産原価/販売) | 約3,885万円/約6,480万円 | 約2,055万円(原価)/約4,000万円 | 約2,925万円/約5,700万円 |
| 閾値マージン | 不足（fusion 75% vs 要求85%） | Phase 2+ PIC: σ_eff 10.8dB(理論)/9.3dB(現実的) vs 閾値7.5dB, +3.3dB(PS)/+0.8dB(全モード理論限界時) | 余裕（QD支援で要件緩和） |
| 冷却要件 | PT冷凍機40kg+2.5kW | なし | PT205 12kg+1.3kW |
| 非ガウス操作 | 不要（本質的にDV） | 未解決（最大課題） | QDが決定論的に供給 |

### 1.2 転換の核心ロジック

```
DV-FBQC方式の構造的課題:
  fusion成功率 75% < 閾値 85% → 閾値未到達
  SNSPD 1,500個 @ 2.5-4.2K → 冷凍機が不可避
  WI-SNSPD成熟確率 55% → Critical Path上の最大リスク

CV-TDM方式の構造的課題:
  非ガウス操作（GKP生成）→ FTQC到達のボトルネック
  光学GKP状態の決定論的生成 → 未解決

ハイブリッドCV+QD方式の解:
  CV方式の室温動作 + 閾値マージン+1.8dB(PS, 現実的)/+3.3dB(PS, 理論限界) → デスクトップ適合性
  QD単一光子（本質的に非ガウス）→ CV方式の弱点を補完
  少数QD（1-4個）→ 均一性問題を回避、クライオ1台に集約
  QD支援により必要スクイージング 3-5dB低減 → 実現性大幅向上
```

## 3. CV計算エンジン詳細

### 3.1 PPLN導波路OPA（bow-tie OPO → PPLN導波路OPA）

> bow-tie PPKTP OPOからPPLN導波路OPAに全面変更。
> サイズ1/120、コスト1/7、アライメントフリー。詳細: `design/02_opa-source.md`

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| 方式 | **MgO:PPLN導波路OPA（単一パス増幅）** | NTT 2025実証 |
| 波長 | 1550nm（テレコム帯） | ホモダイン検出器・ファイバとの親和性 |
| ポンプ | **Option B: 1550nm GS-DFB + SOA/TA + PPLN SHG → 775nm パルス (約150万円)** | Portable: SOA→SHG→1×2, Rack: TA→SHG→1×8 |
| スクイージング（生成） | **≥13dB** | NTT CW 12.1dB実証、SiN clad改良で13dB見込み |
| スクイージング（実効） | **5.0dB (現離散) / 8.5dB (Phase1, 13dB, L=0.39dB) / ≈9.3dB (Phase2+ PIC現実的, L=0.27dB) / 10.8dB (理論限界, L=0.15dB)** | BSモデル V_eff=η×V_sqz+(1−η)+V_nl。旧11.5dBはdB減算による誤り |
| 帯域 | **~500GHz (50mm導波路)** | OPA位相整合帯域、TDMクロック100MHzに対して5000倍以上の余裕 |
| 繰り返し率 | 100MHz | Option B パルスOPA推奨（EO gate除去で0.3dB削減。`02_opa-source.md`） |
| モジュールサイズ | **1U（19インチラック）** | 8本のOPAを1Uに収容 |
| モジュール化 | **ファイバピグテール化、アライメントフリー** | 振動耐性が本質的に高い |
| コスト | **約591万円（8本モジュール）** | 旧bow-tie 約2,400万円から75%削減（SSOT: `02_opa-source.md` §3.2） |

**Phase別PIC移行計画**:
- Phase 0-1 (開始後約9ヶ月-約5年): PPLN導波路OPA 1Uモジュール（13dB安定供給）
- Phase 2 (開始後約5-7年): ホモダインPIC化 + OPAモジュール最適化
- Phase 2.5 (開始後約7-9年): LNOI OPA on-chip（10dB以上達成時）

### 3.2 TDMクラスタ状態生成（macronode lattice具体化）

> 具体的な光学回路・遅延パラメータを確定。詳細: `design/03_tdm-cluster.md`

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| クラスタトポロジー | **macronode lattice (2D)** | Stafford-Menicucci-Walshe 2025で閾値最適化 |
| 生成回路 | **OPA 2本 → BS₁ → 短遅延τ₁ + 長遅延τ₂ → BS₂, BS₃** | Asavanant+ 2019で実証 |
| 短遅延τ₁ | **10ns = PMF 2m** | 1クロック分 |
| 長遅延τ₂ | **980ns = PMF ~200m** | 98列 × 10ns (d=7) |
| モード数 | 時間多重により原理的に無制限 | |
| 物理装置 | **OPA 1U + TDM 2U + 遅延1U + ホモダイン1U = 5U** | |
| ホモダイン | **バランスPD ×2ch (InGaAs, 室温)** | |

**TDMの決定的優位**: 量子ビット数を増やしても物理装置サイズは不変。
10論理qubitでも1,000論理qubitでも同じ20Uラック。

## 4. 非ガウスクライオモジュール詳細

### 4.1 QD光源（非ガウスリソース供給）（InP QD 1550nm直接オプション追加）

**Primary（Go時）: InP QD 1550nm直接発光**

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| QD数 | 1-4個 | 均一性問題を回避する最小数 |
| 材料 | **InAs/InAlGaAs on Si** | arXiv:2602.06140 (2026.02) |
| 波長 | **1550nm（C-band直接）** | QFC完全不要 |
| 不可弁別性 | HOM visibility >0.95（目標） | 実績V=0.90、V>0.95は開始後約1年半で見込み |
| g^2(0) | **<0.01** | 実績0.005 |
| 輝度 | >0.40（目標） | 改善中 |
| 動作温度 | 4K（2.5Kで動作可能） | PT205で対応 |

**Secondary（No-Go時）: GaAs QD 925nm + QFC**

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| QD数 | 1-4個 | 均一性問題を回避する最小数 |
| 材料 | InAs/GaAs（マイクロピラー共振器） | C2N研究所標準品 |
| 波長 | 925nm → QFC → 1550nm | LNOI PPLN on-chip QFC |
| 不可弁別性 | HOM visibility >0.96 | Tomm+ 2021実証済み |
| 輝度 | >0.57 | ファーストレンズ後 |

**Go/No-Go判定 (開始後約18ヶ月)**: InP QD 1550nm単体でV>0.95 + g^2(0)<0.02 + 輝度>0.40の同時達成。
Go確率: 65%。

**InP QD Go時の効果**: QFCモジュール（約600万円 + ポンプレーザ約300万円）が完全不要。
非ガウスモジュールサイズ30%縮小。QFC由来ノイズゼロ→GKP忠実度+0.02向上。

**役割**: QDの決定論的単一光子は**本質的に非ガウスリソース**。
CVクラスタ状態に注入してGKP状態の高品質ヘラルディング生成を支援する。

**Bourassa et al. (2021)の理論**: QD単一光子の使用により、GKP状態生成に必要なスクイージングを
**3-5dB低減**可能。15dB→10-12dBなら、ファイバ化バルクOPOで安定達成。

> **定量的GKP忠実度分析**(レガシー参考): `../archive/03_logical-layer/gkp/01_gkp-design.md` を参照。
> QD 2個使用時のGKP忠実度は0.93（sigma_eff=10dB、損失2dB）と推定。
> Stimシミュレーションで確定予定。
> **注(BSモデル修正後)**: 現行BSモデルではQD=2 Step B F_GKP=0.95（04_gkp-protocol.md §3.3）。上記0.93はdB減算モデルによるレガシー値。

### 4.2 QFC（量子周波数変換）

| パラメータ | 仕様 |
|-----------|------|
| 変換 | 925nm → 1550nm |
| 方式 | LNOI PPLN導波路（on-chip） |
| 効率 | η=0.75-0.82 |
| チャネル数 | 1-4ch（QD数に対応） |

QFC損失は受容可能。非ガウスリソースは計算全体の一部であり、
メインの計算経路（CV 1550nm）には影響しない。

### 4.3 PNR-SNSPD（光子数弁別検出器）

| パラメータ | 仕様 | 根拠 |
|-----------|------|------|
| 数量 | 4-8個 | GKP状態ヘラルディングに必要な最小数 |
| 方式 | マルチピクセルSNSPD | 光子数分解能あり |
| 検出効率 | >95% | QD同一クライオスタット内で最適化 |
| 動作温度 | ~2-4K | QDと同一冷却ステージ |

**DV-FBQC方式との決定的差異**: SNSPD **1,500個→4-8個**（99.5%削減）。
冷凍機の熱負荷が桁違いに小さくなり、小型PT冷凍機で十分。

| 重量 | ~25kg | **8.6kg** | **-66%** |

| 冷却方式 | 空冷 | **空冷（可動部品なし）** | 低振動 |

| コスト | 約600万円 | **約900万円** | +約300万円 |

## 8. リスクと対策

| リスク | 確率 | 影響 | 対策 |
|--------|------|------|------|
| QD支援GKP生成の忠実度不足 | 30% | ハイブリッドの優位性消失 | 純粋CV方式（ヘラルディングGKP）にフォールバック |
| バルクOPOのラック化で安定性劣化 | 25% | スクイージング2-3dB低下 | 密閉筐体+能動位相安定化で対応 |
| CV-DVモード整合の困難 | 35% | QD支援の効率低下 | QFC効率改善、モード整形技術の開発 |
| リアルタイムGKPデコーダの計算負荷 | 20% | レイテンシ超過 | ソフト情報のML近似デコーダ（FPGA実装） |
| DV-FBQCフォールバックの遅延 | 15% | Phase 2.5スケジュール超過 | 並行開発体制の維持（投資の20%をDV路線に配分） |

## Appendix: 設計方針の根拠

### 主要知見

| 分野 | 最重要知見 |
|------|----------|
| CV-TDMアーキテクチャ | CV方式は全系室温動作。閾値マージン+1.8dB(現実的PS)/+3.3dB(理論限界PS) |
| GKP符号・誤り訂正 | 光学GKP実証済み(Campagne-Ibarcq et al. 2020)。10dBで表面符号閾値到達可能 |
| 実装統合 | LNOI上OPO現状3-5dB。ファイバ化→ハイブリッドPIC→フルPICの3段階 |
| 光源（QD vs OPO） | ハイブリッドCV+QD方式。QD 1-4個で非ガウス性を供給 |
| 理論・閾値分析 | CV閾値マージン+1.8〜+3.3dB(PS) > DV閾値未到達。qLDPC+GKPが次世代有望 |
| 工学化・市場 | CV方式はコスト30-60%安。ただしFTQC確実性はDV優位 |

### 設計方針

**ハイブリッドCV+QD方式を主軸、DV-FBQCをフォールバック**
