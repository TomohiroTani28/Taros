## 1. PIC統合ロードマップ

### 1.1 Phase別PIC構成

| Phase | CV計算エンジン | 非ガウスモジュール | 検出系 |
|-------|-------------|----------------|--------|
| Phase 0 (開始後2-3年) | バルクOPO + ファイバBS網 | QD + QFC (ディスクリート) | バルクホモダイン |
| Phase 1 (開始後4-6年) | バルクOPO密閉モジュール | QD + on-chip QFC | **ホモダインPIC** |
| Phase 2 (開始後6-8年) | **OPO一部PIC化** | QD統合モジュール | ホモダインPIC |
| **Phase 2.5 (開始後8-10年)** | **OPO全PIC化** | **PT205統合モジュール** | **統合PIC** |

### 1.2 DV-FBQC PICとの共通技術

`../archive/01_architecture/06_integrated-photonics-roadmap.md` で設計済みの技術のうち、
ハイブリッド方式でも活用可能なもの:

| 技術 | DV-FBQCでの用途 | CV+ハイブリッドでの用途 | 共通度 |
|------|---------------|---------------------|--------|
| LNOI EO変調器 | fusionゲート制御 | **位相変調・スイッチング** | 100% |
| SiN導波路 | 受動光回路 | **BS網・遅延線** | 100% |
| LNOI+SiNヘテロジニアス集積 | 統合PIC | **統合PIC** | 100% |
| Transfer Printing | QDチップレット搭載 | **QDチップレット搭載** | 100% |
| WI-SNSPD | 大規模検出アレイ | **PNR-SNSPD 4-8個** | 製造プロセス共通 |
| Flip-Chip Bonding (C33) | LNOIチップレット | **LNOIチップレット** | 100% |

**共通度は極めて高い。DV-FBQC基盤のPIC投資はハイブリッド方式でもほぼ全額活用可能。**

## 2. CV計算エンジンPIC

### 2.1 LNOI OPO on-chip

Phase 2.5の目標: PPLN導波路OPA（現主設計）をLNOI PIC上のOPOに置換。
**注 (v2.2)**: 下表の「バルクOPO (現在)」列は旧設計（bow-tie OPO）のスペック。
現主設計はPPLN導波路OPA (13dB, `design/02_opa-source.md` 参照)。

| パラメータ | バルクOPO (現在) | **LNOI OPO (目標)** |
|-----------|----------------|-------------------|
| スクイージング | 15dB (実験室) / 12dB (製品) | **10-12dB (Phase 2.5)** |
| サイズ | 30cm x 20cm x 10cm | **20mm x 5mm** |
| 安定性 | 能動位相ロック必要 | **モノリシック、自己安定** |
| 消費電力 | ポンプ100mW + ロック10mW | ポンプ50mW |
| コスト | 約300万円/台 | **約30万円/チップ** |
| 問題点 | — | **LNOI上で10dB達成は未実証 (現状3-5dB)** |

**Critical Gap: LNOI OPOのスクイージングを3-5dB→10dB に改善する必要がある。**

**注意 (v2.0 訂正)**: dB減算モデル（σ_eff = OPO − L）は物理的に不正確。
正しいビームスプリッタ損失モデルでは、OPO=13dB・L_total=1.53dBでも
σ_eff≈5.0dBにしかならず閾値未達。PIC統合により L_total≤0.15dB を達成することが必須。
詳細は `design/06_noise-budget.md` を参照。
LNOI 10-12dBではPIC低損失化なしにLevel A/B いずれも未達。
13dB未満の場合はバルクPPLN OPA継続がフォールバック。

改善ロードマップ:
1. 開始後約2年: LNOI OPO 7dB (導波路長最適化 + Q値向上)
2. 開始後約4年: LNOI OPO 9dB (結合損失低減 + フォトリフラクティブ対策)
3. 開始後約6年: LNOI OPO 10-12dB (マイクロリング共振器 + cryogenic動作オプション)

10dB未達の場合: バルクOPO密閉モジュールをラック内に維持（Phase 1構成を継続）。

### 2.2 BS網 + 位相シフタ PIC

TDMクラスタ状態生成に必要な光学要素:

| 要素 | 数量 | PIC実装 | 損失 |
|------|------|---------|------|
| 50:50 ビームスプリッタ | 8-16 | SiN方向性結合器 | 0.02dB/個 |
| 位相シフタ | 8-16 | LNOI EO | 0.05dB/個 |
| 光スイッチ (2x2) | 4-8 | LNOI MZI | 0.3dB/個 |
| 遅延線 (短) | 2-4 | SiNスパイラル (1-10m) | 0.01dB/m |
| 遅延線 (長) | 2 | 外部PMF (10-100m) | 0.18dB/km |

**PIC損失合計: ~1.5dB**
(BS 0.3dB + EO 0.8dB + スイッチ 0.3dB + その他 0.1dB)

| パラメータ | バルク (現在) | **PIC (Phase 1+)** |

## 3. 非ガウスクライオモジュール PIC

### 3.1 PT205統合モジュール設計

```
PT205コールドヘッド (2.5K)
    |
    └── 非ガウスPIC (15mm x 5mm)
        ├── [InP QDチップレット x 2] (Transfer Printing)
        │     1550nm直接発光、QFC不要
        │
        ├── [SiN導波路] (光子ルーティング)
        │     QD出力→CVクラスタへの注入経路
        │
        ├── [WI-SNSPD コム型 x 4-8] (膜転写)
        │     PNR対応 (2段カスケード)
        │     GKP状態ヘラルディング用
        │
        └── [LNOI EO位相シフタ x 4]
              QD光子の位相制御
              CV光との干渉制御
```

### 3.2 室温PICとクライオPIC間の接続

| 接続 | 方式 | 本数 | 損失 |
|------|------|------|------|
| QD光子出力 → CVクラスタ | PMFファイバ | 2-4 | 0.3dB/接続 |
| LO光入力 → ホモダイン基準 | SMFファイバ | 2 | 0.1dB/接続 |
| SNSPD読み出し | RF同軸 (MKID) | 2 | — |
| DC制御 | DC配線 | ~10 | — |
| **合計** | | **~18本** | |

## 5. 技術リスクと判定

| リスク | 確率 | 影響 | Phase 2.5フォールバック |
|--------|------|------|---------------------|
| LNOI OPO <10dB | 45% | バルクOPO継続 (+6U, +約1,500万円) | Phase 1構成をラック内に維持 |
| ホモダインPIC量子効率<97% | 20% | バルクホモダイン継続 (+2U) | 性能影響小 |
| InP QD 1550nm No-Go | 35% | GaAs QD + on-chip QFC | +約600万円, 非ガウスPIC複雑化 |
| PT205内PIC CTE問題 | 25% | SiO2緩衝層 (C34対策) | 設計済み対策適用 |
